Quizora (Quiz App)

A modern, interactive quiz application built using React.js and Tailwind CSS, featuring category-based quizzes, timer system, result analysis, and a local leaderboard system.

✨ Features
🎯 Category-based quizzes (Tech, Movies, General, Fun, etc.)
⏱️ 15-second timer per question
📊 Progress tracking (2/5, 3/5 format)
🏆 Result screen with score percentage
💬 Dynamic feedback messages based on performance
📈 Local leaderboard system (stored in localStorage)
🔄 Try Again & Restart quiz option
🏠 Smooth navigation (Home, Quiz, Result, Leaderboard)
🎨 Modern UI with glassmorphism + gradient design
✨ Animated background effects

🛠️ Tech Stack
React.js ⚛️
React Router DOM
Tailwind CSS 🎨
JavaScript (ES6+)
LocalStorage (for leaderboard storage)

🧩 Key Functionalities
🎮 Quiz System
Questions load based on selected category
Auto move to next question after selection or timer ends
⏱️ Timer
15 seconds per question
Auto submit when time runs out
🏆 Leaderboard
Stores scores in localStorage
Sorted by highest score
Option to clear leaderboard
📊 Result System
Shows percentage score
Displays performance message:
100% → Excellent
80%+ → Great job
50%+ → Good effort
<50% → Try again
